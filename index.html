Projects

Pooled Time Series

Betterment of an existing analysis of video files in terms of similarity analysis through sequencing of certain types of information across frames.

While establishing the concept, the original analysis was not capable of comparing multiple videos (in the magnitude of thousands) for similarity analysis.

This project makes use of OpenCV Java library to read the videos and track the movement within the videos by breaking them into frames.

Also uses Hadoop Map Reduce in order to ensure faster processing as well as prevent the program from running out of memory (which was one of the original issue) as there were more than 6800 videos to be compared.

Evaluating Content Analysis for TREC dataset

The refined TREC data set was used to evaluate the following:

  1. Identify the classification path from request to content made by the crawler
  2. Derive the file size diversity according to MIME type
  3. Demonstrate Tika Parser Call Chain according to MIME type
  4. Compute the Language Diversity for the Dataset
  5. Evaluate the maximal joint agreement between NLTK, CoreNLP/NER, OpenNLP

This was done using Apache Tika. The visualizations were produced with the help of D3.js
